40 signets Semaine du 2026-03-27

Synthese de la semaine

1. Claude Code : l'ecosysteme explose

La majorite des signets tournent autour de Claude Code, qui s'impose comme la plateforme centrale du dev assiste par IA. Trois axes majeurs :

- Taches cloud recurrentes (`/schedule`) : Claude Code devient un agent permanent, capable de tourner en autonomie pour reviewer des PRs, maintenir la doc ou corriger des echecs CI — sans machine locale allumee.

- Multi-agents : Anthropic publie officiellement son architecture 3 agents (orchestrateur/executeur/verificateur) pour le coding longue duree. En parallele, des outils communautaires comme `claude-peers-mcp` permettent a plusieurs instances de collaborer localement.

- Personnalisation poussee : Skills sur mesure (`/interview`, `/plan-ceo-review` via gstack de Garry Tan), hooks pre-commit intelligents (`/simplify` + `/security-review`), commande `/model opusplan` pour optimiser cout/performance. L'outil se transforme en IDE IA configurable a l'infini.

2. Frontend et design : la convergence dev/design s'accelere

- Figma MCP ouvre le canvas Figma aux agents IA avec acces au design system — plus besoin de decrire les composants, Claude les manipule directement.

- Selection DOM dans Claude Code Desktop : pointer un element dans le navigateur au lieu de le decrire. React detecte automatiquement le composant source.

- Google Stitch : generation de landing pages completes en 10 minutes, gratuitement. Le prototypage frontend devient quasi-instantane.

3. SEO automatise : un theme recurrent et monetisable

Trois signets distincts couvrent l'automatisation SEO via Claude : generation de pages WordPress optimisees, 9 prompts prets a l'emploi pour remplacer une agence, et des tutoriels concrets. Le message est clair : le SEO programmatique assiste par IA est le cas d'usage business le plus immediatement monetisable.

4. Productivite et outillage peripherique

- Monitoring : plugin statusline terminal, app macOS barre de menu pour suivre sa consommation de tokens.

- Memoire persistante : Claude-Mem promet 95% de tokens economises en conservant le contexte entre sessions.

- Terminal IA : Kaku, fork de WezTerm, corrige les erreurs de commande via l'IA.

- Dispatch : piloter Claude depuis son telephone pendant qu'il travaille sur votre machine — le dev asynchrone devient realite.

5. Formation et ressources officielles

- Academie Anthropic gratuite : 13 cours + certifications sur MCP, API et Claude Code.

- Cheat sheet de 33 pages : guide officiel pour designer et construire avec Claude (60K+ bookmarks).

- Tips du createur Boris Cherny : 104K bookmarks, 9M vues — le thread de reference.

6. Transparence et confiance dans les modeles

- Fuite "Mythos" : un modele Anthropic superieur a Opus serait en preparation, freine par les risques cyber et le cout d'inference. A confirmer.

- Scandale Cursor : Composer 2, presente comme un modele maison, serait un rebadge de Kimi K2.5. Rappel salutaire sur la transparence des benchmarks proprietaires.
