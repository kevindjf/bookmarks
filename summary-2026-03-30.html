40 signets Semaine du 2026-03-30

Synthese de la semaine

1. Claude Code devient un agent autonome permanent

La tendance dominante : Claude Code evolue d'un outil interactif vers un systeme d'agents autonomes. Trois annonces convergent — les taches recurrentes cloud (`/schedule`), l'architecture 3 agents (orchestrateur/executeur/evaluateur) publiee par Anthropic, et Dispatch qui permet de lancer des sessions depuis mobile et les retrouver terminees. Le message est clair : l'IA code pendant que vous dormez.

2. L'ecosysteme de plugins et skills explose

Une effervescence autour de l'extensibilite de Claude Code : gstack (setup du CEO de YC en 6 specialistes), claude-peers-mcp (communication entre instances), Claude-Mem (memoire persistante, -95% tokens), statusline terminal, app de suivi de consommation, et des pre-commit hooks automatisant simplification, securite et tests. Claude Code n'est plus un produit — c'est une plateforme.

3. Le design et le frontend sous controle IA

Figma lance son MCP en open beta : les agents designent directement sur le canvas. Claude Code Desktop permet la selection d'elements DOM en un clic. Google entre dans la course avec Stitch pour generer des landing pages. Le workflow design-to-code se compresse radicalement.

4. SEO automatise par IA : un cas d'usage en plein essor

Plusieurs signets convergent sur l'utilisation de Claude Code pour generer des pages SEO a la chaine connectees a WordPress, avec des collections de prompts dediees. Le SEO devient un workflow quasi-industrialise par agents.

5. Formation et documentation officielles

Anthropic structure l'adoption : academie gratuite (13 cours + certifications), cheat sheet de 33 pages pour designer avec Claude, et un guide de creation de skills. L'investissement dans l'education signale une volonte de creer un ecosysteme de praticiens certifies.

6. Transparence et course aux modeles

La fuite du modele "Mythos" (au-dessus d'Opus, potentiellement dangereux) et la controverse Cursor/Kimi K2.5 rappellent deux realites : les capacites avancent plus vite que les publications, et la transparence sur les modeles utilises reste un enjeu critique.

---

Insight cle : On assiste a la transition de l'IA coding d'outil a infrastructure. Les briques sont en place — agents autonomes, communication inter-agents, taches planifiees, integrations design, memoire persistante. Le differenciateur n'est plus l'acces au modele mais la qualite de l'orchestration qu'on construit autour.
