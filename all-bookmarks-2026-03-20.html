25 signets IA 2021 - 2026

Synthèse

1. Claude Code : l'écosystème dominant

La majorité des signets (16/25) gravitent autour de Claude Code, révélant un écosystème en pleine maturation :

- Workflows et planification : Mode `opusplan` (Opus pour réfléchir, Sonnet pour exécuter), règles de planification obligatoire, skill `/interview` pour forcer la réflexion avant l'action. Le message est clair : la qualité vient de la discipline, pas de la vitesse.

- Skills et extensibilité : gstack de Garry Tan, pre-commit hooks automatisés (/simplify, /security-review), plugins communautaires. Claude Code se transforme en plateforme orchestrant une équipe de spécialistes.

- Mémoire et persistance : Claude-Mem (réduction de 95% des tokens), Cowork Dispatch (assistant asynchrone accessible depuis mobile). Le problème de la continuité entre sessions est activement résolu.

- Ressources officielles : Cheat sheet Anthropic de 33 pages, académie gratuite avec 13 cours certifiants. Anthropic investit massivement dans l'adoption et la formation.

Insight clé : Les power users convergent vers un setup minimaliste (cf. Boris Cherny) enrichi de quelques skills ciblées, plutôt que vers une sur-configuration.

---

2. Agents autonomes et automatisation business

- Paperclip + Claude : entreprise zéro humain tournant en local

- Agent Teams : orchestration multi-agents visualisable

- SEO automatisé : génération de pages WordPress à la chaîne, 9 prompts SEO dédiés

La tendance de fond : l'IA passe de l'assistant ponctuel à l'agent autonome capable de gérer des workflows complets (vente, contenu, opérations).

---

3. Outillage périphérique

- Kaku : terminal macOS pensé pour le dev assisté par IA

- App usage Claude Code : monitoring des quotas en barre de menu

- CLI signets Twitter : veille automatisée sans API payante

Un écosystème d'outils satellites émerge pour combler les frictions autour de l'usage intensif des LLM.

---

4. Usages grand public de l'IA

Trois signets sortent du cadre dev :

- Apprentissage des langues via prompts structurés

- Planification de repas hebdomadaires

- Curation d'outils IA spécialisés

Ces cas illustrent la démocratisation des LLM comme assistants du quotidien.

---

Tendances transversales

| Tendance | Signal |

|---|---|

| Planifier avant d'exécuter | Skills /interview, mode plan obligatoire, opusplan |

| Automatisation en pipeline | Pre-commit hooks, SEO en série, agents autonomes |

| Persistance et asynchronie | Claude-Mem, Cowork Dispatch, mémoire inter-sessions |

| Communauté > documentation | Les tips des créateurs et power users génèrent 100k+ bookmarks |

| Claude Code comme plateforme | Plugins, skills, hooks, agent teams — plus un IDE agent qu'un simple CLI |

En une phrase : L'écosystème Claude Code est en train de passer du stade "outil de dev assisté" à celui de plateforme d'orchestration d'agents, et la communauté francophone est particulièrement active dans cette adoption.
