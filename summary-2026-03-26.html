40 signets Semaine du 2026-03-26

Synthese de la semaine

1. Claude Code : l'ecosysteme explose

La majorite des signets tournent autour de Claude Code, qui s'impose comme la plateforme centrale du dev assiste par IA. Trois axes majeurs :

- Taches cloud recurrentes (`/schedule`) : Claude Code peut desormais tourner en autonomie dans le cloud — review de PRs, maintenance de docs, triage d'issues — sans machine locale active. C'est le passage du copilote a l'agent autonome.

- Multi-agents et orchestration : Architecture a 3 agents (orchestrateur/executeur/verificateur) documentee par Anthropic, visualisation des Agent Teams, communication inter-instances via `claude-peers-mcp`. Le paradigme passe du "un agent, une tache" au "une equipe d'agents coordonnes".

- Personnalisation avancee : Skills custom (`/interview`, gstack de Garry Tan), pre-commit hooks intelligents (`/simplify` + `/security-review`), mode `opusplan` (Opus planifie, Sonnet execute), nouveau `/init` interactif. Claude Code devient un framework qu'on configure a son image.

2. Design et frontend : l'IA entre dans Figma

Convergence nette entre IA et outils de design :

- MCP Figma permet a Claude de designer directement sur le canvas avec connaissance du design system.

- Claude Code Desktop offre la selection DOM par clic — plus besoin de decrire, on pointe.

- Google Stitch genere des landing pages completes gratuitement.

Le message est clair : le mur entre designer et developpeur s'effondre.

3. SEO automatise : la production de contenu a l'echelle

Plusieurs signets convergent vers l'utilisation de Claude pour le SEO programmatique : generation de pages WordPress optimisees, 9 prompts avances pour remplacer une agence SEO, production en masse de contenu positionne. L'IA devient une machine a trafic organique pour qui maitrise les prompts.

4. Outillage et productivite dev

Un ecosysteme d'outils peripheriques se structure :

- Kaku : terminal macOS avec IA integree (correction d'erreurs automatique)

- Claude-Mem : memoire persistante entre sessions (-95% de tokens)

- App de suivi de quotas macOS pour monitorer sa consommation

- Plugin statusline pour visualiser l'activite en temps reel

- CLI de signets Twitter sans API payante pour alimenter sa veille

5. Formation et ressources officielles

Anthropic investit massivement dans l'education :

- Academie gratuite : 13 cours avec certifications (MCP, API, Claude Code)

- Cheat sheets de 33 pages : une pour le design/build, une pour la creation de skills

- Conseils du createur Boris Cherny (104k bookmarks — le tweet de reference)

6. Transparence et vigilance

Le scandale Cursor/Kimi K2.5 rappelle que les benchmarks marketing ne valent rien sans verification independante. Un signal d'alerte utile dans un marche ou chaque outil revendique la suprematie.

---

Tendance dominante : On passe de "l'IA qui assiste" a "l'IA qui opere" — agents autonomes, taches planifiees, orchestration multi-agents, design automatise. Claude Code n'est plus un outil, c'est une plateforme sur laquelle on deploie des equipes d'agents specialises. Ceux qui investissent maintenant dans la configuration (skills, hooks, workflows) prendront une avance considerable.
