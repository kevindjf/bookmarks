40 signets Semaine du 2026-03-25

Synthese de la semaine

1. Claude Code devient une plateforme d'agents autonomes

La tendance dominante : Claude Code n'est plus un simple assistant de coding. Avec les taches programmees cloud (`/schedule`), l'architecture multi-agents (orchestrateur/executeur/verificateur), et Dispatch (conversations persistantes accessibles depuis mobile), Anthropic construit un ecosysteme d'agents qui tournent 24/7 sans supervision. Le projet claude-peers-mcp pousse encore plus loin avec la communication inter-agents en local.

2. Le workflow "planifier avant de coder" s'impose

Plusieurs signets convergent : le plan mode, le skill `/interview`, le `/plan-ceo-review` de Garry Tan (gstack), et le mode opusplan (Opus planifie, Sonnet execute). Le message est clair — les utilisateurs avances ne laissent plus Claude foncer tete baissee. La discipline planification/execution/verification devient la norme.

3. L'integration design/dev franchit un cap

Figma MCP permet aux agents de designer directement sur le canvas avec acces au design system. Claude Code Desktop ajoute la selection DOM visuelle (clic sur un element = contexte complet envoyé a Claude, y compris composant React et props). Google Stitch arrive en concurrent gratuit pour le prototypage de landing pages. Le mur entre design et code s'effondre.

4. Skills, hooks et configuration : l'industrialisation de Claude Code

Le guide Anthropic de 33 pages, le setup open source gstack de Garry Tan, les hooks pre-commit (simplify + security-review + tests auto), et le nouveau /init interactif montrent une communaute qui structure et industrialise ses workflows. Les tips de Boris Cherny (createur de Claude Code) tempèrent : un setup minimaliste reste parfaitement viable.

5. SEO automatise par IA : une niche en pleine explosion

Trois signets distincts sur la generation de pages SEO en masse via Claude Code + WordPress, plus 9 prompts SEO prets a l'emploi en mode Cowork. Le pattern est simple : mots-cles → generation → publication. Un cas d'usage concret ou l'IA remplace directement le travail d'agence.

6. Outillage et productivite

- Claude-Mem : memoire persistante entre sessions (95% de tokens economises)

- Kaku : terminal macOS optimise pour le dev IA

- App macOS de suivi de quota : indispensable pour les utilisateurs intensifs

- CLI signets Twitter : veille automatisee sans API payante

- Visualisation Agent Teams : transparence sur l'orchestration multi-agents

7. Formation et transparence

L'academie gratuite Anthropic (13 cours + certifications) democratise l'acces aux competences MCP/API/Claude Code. Cote transparence, le scandale Cursor/Kimi K2.5 rappelle de toujours verifier l'origine des modeles derriere les outils.
